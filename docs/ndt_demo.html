高斯怎么计算

μ = (p₁ + ... + pₙ) / n
Σ = ∑(pᵢ - μ)(pᵢ - μ)T / (n - 1)

绿色地图高斯：先验地图点按 voxel 分组后，对每个格子里的点算均值 μ 和协方差 Σ。 橙色源高斯：实时点云先用当前位姿 T 变到地图坐标系，再按同样方法分格子计算，用来演示 D2D-NDT。

标准 NDT 评分

q = T p
d² = (q - μ)T Σ-1 (q - μ)
score = exp(-0.5 d²)

q 是实时点变换到地图坐标系后的位置；μ 和 Σ 来自 q 所在地图格子的高斯。 d² 越小，点越落在高概率区域，整体位姿越可信。
